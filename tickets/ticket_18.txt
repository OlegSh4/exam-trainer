## Словарь в Python
Словарь в Python — это структура данных, которая хранит пары вида “ключ — значение”.
Простыми словами, словарь похож на обычный словарь: есть слово, и есть его значение.
В Python это выглядит так:
$$
key \to value
$$
То есть каждому ключу соответствует некоторое значение.
Например, можно хранить данные о студенте:
```python
student = {
    "name": "Oleg",
    "age": 22,
    "group": "PM22-3"
}

Здесь "name", "age", "group" — это ключи.

А "Oleg", 22, "PM22-3" — это значения.

Назначение словарей

Словари нужны, когда данные удобно хранить не по номеру позиции, как в списке, а по имени ключа.

Например, в списке:

student = ["Oleg", 22, "PM22-3"]

не сразу понятно, что означает каждый элемент.

А в словаре:

student = {
    "name": "Oleg",
    "age": 22,
    "group": "PM22-3"
}

сразу видно, где имя, где возраст, где группа.

Словари часто используются для хранения:

данных пользователя;

настроек программы;

результатов подсчета;

частот слов;

JSON-данных;

информации из базы данных;

параметров модели.

Основные особенности словарей

Словарь хранит данные в виде пар:

ключ: значение

Например:

person = {
    "name": "Anna",
    "age": 20
}

Ключи в словаре должны быть уникальными.

Если один и тот же ключ указать несколько раз, сохранится последнее значение:

data = {
    "x": 10,
    "x": 20
}

В результате в словаре будет:

{"x": 20}

Ключи должны быть неизменяемыми объектами. Обычно ключами бывают строки, числа или кортежи.

Например:

data = {
    "name": "Ivan",
    1: "first",
    (2, 3): "point"
}

Значениями могут быть любые объекты: строки, числа, списки, другие словари, функции и так далее.

Отличие словаря от списка

Список хранит элементы по индексам.

Например:

numbers = [10, 20, 30]

Чтобы получить первый элемент, пишут:

numbers[0]

Словарь хранит элементы по ключам.

Например:

student = {
    "name": "Oleg",
    "age": 22
}

Чтобы получить имя, пишут:

student["name"]

Главное отличие:

в списке доступ идет по номеру позиции;

в словаре доступ идет по ключу.

Создание словаря

Словарь можно создать с помощью фигурных скобок:

student = {
    "name": "Oleg",
    "age": 22,
    "group": "PM22-3"
}

Пустой словарь создается так:

data = {}

Также словарь можно создать с помощью функции dict():

student = dict(name="Oleg", age=22, group="PM22-3")

Результат будет таким:

{
    "name": "Oleg",
    "age": 22,
    "group": "PM22-3"
}

Создание словаря из пар

Словарь можно создать из списка пар.

Например:

pairs = [
    ("name", "Oleg"),
    ("age", 22),
    ("group", "PM22-3")
]
student = dict(pairs)

Результат:

{
    "name": "Oleg",
    "age": 22,
    "group": "PM22-3"
}

Это удобно, когда данные приходят в виде пар “ключ — значение”.

Получение значения по ключу

Чтобы получить значение из словаря, нужно обратиться к нему по ключу.

Например:

student = {
    "name": "Oleg",
    "age": 22,
    "group": "PM22-3"
}
print(student["name"])

Результат:

Oleg

Если обратиться к ключу, которого нет, возникнет ошибка:

print(student["city"])

Ошибка будет типа KeyError.

Метод get

Чтобы безопасно получить значение по ключу, используют метод get().

Пример:

student = {
    "name": "Oleg",
    "age": 22
}
city = student.get("city")
print(city)

Результат:

None

Если ключа нет, метод get() возвращает None, а не вызывает ошибку.

Можно указать значение по умолчанию:

city = student.get("city", "Moscow")
print(city)

Результат:

Moscow

Простыми словами, get() удобен, когда мы не уверены, что ключ есть в словаре.

Добавление нового элемента

Чтобы добавить новый элемент в словарь, нужно указать новый ключ и значение.

Например:

student = {
    "name": "Oleg",
    "age": 22
}
student["group"] = "PM22-3"

Теперь словарь будет таким:

{
    "name": "Oleg",
    "age": 22,
    "group": "PM22-3"
}

Если ключа раньше не было, он добавится.

Изменение значения

Чтобы изменить значение по существующему ключу, нужно присвоить новое значение.

Например:

student = {
    "name": "Oleg",
    "age": 22
}
student["age"] = 23

Теперь значение по ключу "age" изменится:

{
    "name": "Oleg",
    "age": 23
}

Если ключ уже есть, значение обновляется.

Если ключа нет, создается новая пара.

Удаление элементов

Удалить элемент из словаря можно с помощью оператора del.

Пример:

student = {
    "name": "Oleg",
    "age": 22,
    "group": "PM22-3"
}
del student["age"]

После удаления:

{
    "name": "Oleg",
    "group": "PM22-3"
}

Также можно использовать метод pop().

age = student.pop("age")

Метод pop() удаляет элемент и возвращает его значение.

Если ключа нет, будет ошибка. Но можно указать значение по умолчанию:

city = student.pop("city", "not found")

Если ключа "city" нет, вернется "not found".

Метод clear

Метод clear() удаляет все элементы словаря.

Пример:

student = {
    "name": "Oleg",
    "age": 22
}
student.clear()

После этого словарь станет пустым:

{}

Проверка наличия ключа

Чтобы проверить, есть ли ключ в словаре, используют оператор in.

Пример:

student = {
    "name": "Oleg",
    "age": 22
}
print("name" in student)

Результат:

True

Проверка:

print("city" in student)

Результат:

False

Оператор in проверяет именно ключи, а не значения.

Длина словаря

Количество пар “ключ — значение” можно узнать с помощью функции len().

Пример:

student = {
    "name": "Oleg",
    "age": 22,
    "group": "PM22-3"
}
print(len(student))

Результат:

3

Это означает, что в словаре три пары.

Основные методы словарей

Метод keys() возвращает все ключи словаря.

student.keys()

Метод values() возвращает все значения словаря.

student.values()

Метод items() возвращает пары “ключ — значение”.

student.items()

Метод get() безопасно получает значение по ключу.

student.get("name")

Метод pop() удаляет элемент по ключу и возвращает его значение.

student.pop("age")

Метод update() добавляет или обновляет элементы словаря.

student.update({"city": "Moscow", "age": 23})

Метод clear() очищает словарь.

student.clear()

Метод update

Метод update() используется для добавления или обновления сразу нескольких элементов.

Пример:

student = {
    "name": "Oleg",
    "age": 22
}
student.update({
    "age": 23,
    "group": "PM22-3"
})

После выполнения:

{
    "name": "Oleg",
    "age": 23,
    "group": "PM22-3"
}

Ключ "age" уже был, поэтому его значение обновилось.

Ключа "group" не было, поэтому он добавился.

Итерация по словарю

Итерация — это перебор элементов.

Если написать цикл по словарю напрямую, будут перебираться ключи.

Пример:

student = {
    "name": "Oleg",
    "age": 22,
    "group": "PM22-3"
}
for key in student:
    print(key)

Результат:

name
age
group

То есть по умолчанию цикл проходит по ключам.

Итерация по ключам

Можно явно использовать метод keys():

for key in student.keys():
    print(key)

Результат будет таким же:

name
age
group

Обычно можно писать просто:

for key in student:
    print(key)

Потому что словарь и так перебирается по ключам.

Итерация по значениям

Чтобы перебрать только значения, используют метод values().

Пример:

for value in student.values():
    print(value)

Результат:

Oleg
22
PM22-3

Это удобно, если ключи не нужны, а нужны только значения.

Итерация по парам ключ — значение

Чаще всего используют метод items().

Он позволяет одновременно получить ключ и значение.

Пример:

for key, value in student.items():
    print(key, value)

Результат:

name Oleg
age 22
group PM22-3

Это самый удобный способ перебора словаря, когда нужны и ключи, и значения.

Вложенные словари

Значением в словаре может быть другой словарь.

Например:

students = {
    "student_1": {
        "name": "Oleg",
        "age": 22
    },
    "student_2": {
        "name": "Anna",
        "age": 21
    }
}

Чтобы получить имя первого студента:

students["student_1"]["name"]

Результат:

Oleg

Вложенные словари часто используются для хранения сложных данных.

Словари и JSON

Словари в Python очень похожи на формат JSON.

Например:

user = {
    "id": 1,
    "name": "Oleg",
    "is_active": True
}

Такие структуры часто используются при работе с API, веб-приложениями и базами данных.

Например, сервер может вернуть данные пользователя в формате JSON, а в Python эти данные удобно обрабатывать как словарь.

Подсчет частот с помощью словаря

Словари часто используют для подсчета количества элементов.

Например, нужно посчитать, сколько раз каждое слово встречается в списке:

words = ["cat", "dog", "cat", "bird", "dog", "cat"]
freq = {}
for word in words:
    if word in freq:
        freq[word] += 1
    else:
        freq[word] = 1

Результат:

{
    "cat": 3,
    "dog": 2,
    "bird": 1
}

Здесь ключ — это слово, а значение — количество его появлений.

Словарное включение

Словарь можно создать коротко с помощью словарного включения.

Например, создадим словарь квадратов чисел:

squares = {x: x**2 for x in range(1, 6)}

Результат:

{
    1: 1,
    2: 4,
    3: 9,
    4: 16,
    5: 25
}

Это похоже на генератор списков, но создает словарь.

Изменяемость словаря

Словарь является изменяемой структурой данных.

Это значит, что после создания словаря можно добавлять, изменять и удалять элементы.

Например:

data = {"a": 1}
data["b"] = 2
data["a"] = 10
del data["b"]

После этих операций словарь изменится.

Это отличает словари от неизменяемых типов, например кортежей.

Порядок элементов в словаре

В современных версиях Python словари сохраняют порядок добавления элементов.

Например:

data = {}
data["a"] = 1
data["b"] = 2
data["c"] = 3

При переборе элементы будут идти в порядке добавления:

a
b
c

Но главное назначение словаря все равно не порядок, а быстрый доступ к значению по ключу.

Быстрый доступ по ключу

Одно из главных преимуществ словаря — быстрый доступ к значению по ключу.

Например:

student["name"]

обычно выполняется очень быстро, даже если в словаре много элементов.

Поэтому словари удобны для поиска данных по идентификатору, имени, коду, номеру или другому ключу.

Например:

users = {
    101: "Oleg",
    102: "Anna",
    103: "Ivan"
}

Чтобы быстро найти пользователя с id 102:

users[102]

Результат:

Anna

Типичные ошибки при работе со словарями

Первая ошибка — обращение к несуществующему ключу:

student["city"]

Если ключа "city" нет, возникнет KeyError.

Чтобы избежать ошибки, лучше использовать:

student.get("city")

Вторая ошибка — попытка использовать изменяемый объект как ключ.

Например, список нельзя использовать как ключ:

data = {[1, 2]: "value"}

Такой код вызовет ошибку, потому что список изменяемый.

Можно использовать кортеж:

data = {(1, 2): "value"}

Третья ошибка — путать ключи и значения при проверке через in.

Выражение:

"name" in student

проверяет наличие ключа "name", а не значения.

Краткий ответ для экзамена

Словарь в Python — это изменяемая структура данных, которая хранит пары “ключ — значение”.

Словарь создается с помощью фигурных скобок:

student = {
    "name": "Oleg",
    "age": 22,
    "group": "PM22-3"
}

Ключи в словаре должны быть уникальными и неизменяемыми объектами. Обычно ключами являются строки, числа или кортежи. Значениями могут быть любые объекты: числа, строки, списки, другие словари и так далее.

Доступ к значению выполняется по ключу:

student["name"]

Если ключа нет, возникнет ошибка KeyError. Для безопасного доступа используют метод:

student.get("name")

Добавление и изменение элементов выполняется через присваивание:

student["age"] = 23
student["city"] = "Moscow"

Удалять элементы можно с помощью del или метода pop():

del student["age"]
student.pop("age")

Основные методы словаря:

keys() — возвращает ключи;

values() — возвращает значения;

items() — возвращает пары “ключ — значение”;

get() — безопасно получает значение по ключу;

update() — добавляет или обновляет элементы;

pop() — удаляет элемент и возвращает его значение;

clear() — очищает словарь.

Итерироваться по словарю можно несколькими способами.

По ключам:

for key in student:
    print(key)

По значениям:

for value in student.values():
    print(value)

По парам “ключ — значение”:

for key, value in student.items():
    print(key, value)

Словари применяются для хранения структурированных данных, настроек, результатов подсчета, JSON-данных, информации о пользователях и быстрого поиска значений по ключам.
