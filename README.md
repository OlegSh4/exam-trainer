# ЭкзаменАИ — Тренажёр к госэкзаменам

Локальное веб-приложение для подготовки к госэкзаменам с использованием Mistral AI.

## Быстрый старт

```bash
# 1. Установить зависимости
pip install -r requirements.txt

# 2. (Опционально) задать API-ключ через окружение
export MISTRAL_API_KEY=your_key_here

# 3. Запустить
./start.sh
# или
uvicorn app:app --reload --port 8000

# 4. Открыть в браузере
# http://localhost:8000
```

## Структура папки билетов

Поместите файлы `.txt` или `.md` в папку `tickets/`:

```
tickets/
  ticket_01.txt
  ticket_02.txt
  ...
```

### Формат билета

```
Билет №N: Название темы

## Подраздел 1
Текст объяснения с формулой $E = mc^2$.

## Подраздел 2
Многострочная формула:
$$\int_a^b f(x)\,dx = F(b) - F(a)$$

Продолжение текста...
```

**Правила:**
- Первая строка — название билета (отображается в меню)
- Формулы в LaTeX: `$...$` для инлайн, `$$...$$` для блочных
- Подзаголовки `##` помогают модели генерировать вопросы по разделам

## Типы вопросов

| Тип | Описание |
|-----|----------|
| 🎯 Все типы | Случайный микс |
| ✍️ Открытые | Свободный текстовый ответ, AI оценивает понимание |
| ☑️ С вариантами | A/B/C/D, один правильный |
| ∫ Формулы | Выбрать правильную формулу из похожих |
| ↔ Сопоставление | Связать формулы с названиями |

## API-ключ Mistral

Получить ключ: https://console.mistral.ai/

Можно задать:
1. В интерфейсе приложения (сохраняется в localStorage)
2. Через переменную окружения `MISTRAL_API_KEY`

## Требования

- Python 3.9+
- Интернет для Mistral API
